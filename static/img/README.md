# Image de couverture

Placez votre image de couverture ici avec le nom : **cover-image.jpg**

L'image sera utilisée comme arrière-plan pour toutes les sections hero (cover) du site :

- Page d'accueil
- Page Services
- Page About Us
- Page Contact
- Page Shop

**Format recommandé :**

- Format : JPG ou PNG
- Nom du fichier : `cover-image.jpg`
- Taille recommandée : 1920x1080 pixels ou plus
- Poids : optimisé pour le web (< 500 KB de préférence)
